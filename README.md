# Genesis IDE

It's a Integrated Development Environment with VSCode + SGDK + Gens + Retroarch. 

### How to start
```shell
docker compose up
```
Then access
[IDE](http://localhost:9090) | [Gens](http://localhost:9000) | [RetroArch](http://localhost:7070)
